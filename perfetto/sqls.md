### 2. 查看特定表的结构

假设你发现一个表名为 `slice`，可以使用以下查询查看其结构：

```sql
PRAGMA table_info(slice);
```

### 3. 查询表的前几行数据

```sql
SELECT * FROM slice LIMIT 10;
```

通过这些查询，你可以逐步确定 Perfetto 跟踪数据库中的表和列名。以下是一个完整的示例流程：

```sql
-- Step 1: 列出所有表
SELECT name FROM sqlite_master WHERE type='table';

-- 假设你发现一个表名为 slice
-- Step 2: 查看 slice 表的结构
PRAGMA table_info(slice);

-- Step 3: 查询 slice 表的前几行数据
SELECT * FROM slice LIMIT 10;
```
